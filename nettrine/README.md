# Nettrine ORM

This is example of nettrine/orm

## Installation

Run commands:
```
$ composer install
```

Fill DB host, DB name, DB user, DB password and run commands:

Run migrations:
```bash
$ php bin/console migrations:migrate
```

and load sample data:
```bash
$ bin/console doctrine:fixtures:load
```
